포로잭트 소개

###포로잭트 대락적인 예상
상요자의 성별,나이,종교,날씨 등에 따라 관련 음악,옷,음식을 추천하는 앱

###IDE
Android Studio

###Language
JAVA

###Server
Bmob

####세부 사항
1:로그인,로그웃,등록 기능
2:사용자 프로파일 작성,사용자 개인정보 기록
3:items(음악,음식,날씨,옷 등)를 추천
4:SNS
5:베스트 추천
