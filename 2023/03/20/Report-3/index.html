Capstone-Week Three

주요 기능(추천, 공유, 채팅)과 보조 기능(로그인, 가입, 사용자 정보)을 단계별로 구현

EX 추천 기능

1 UI 및 인터페이스 (주좌건)

2 BMOB Database 구축 및 연계 (관지동,주좌건,오성현)

3 API 획득 및 단순화 날씨 API 획득(테스트 단계에서 무료 버전 사용) 획득한 API를 ArrayList 또는 MAP으로 만들어 해시태그 만들기 (관지동)

4 Function code 추천을 예로 들자면 먼저 일정량의 데이터를 hastag 태그로 BMOB DB에 저장하고 현재 날씨 API에서 획득한 hastag에 따라 임의로 10개의 데이터를 뽑아서 표시한다. (관지동,오성현)

5 통합 (오성현) 테스트 (관지동)

5 인터페이스 설계 및 코드 최적화 (관지동,주좌건,오성현)
